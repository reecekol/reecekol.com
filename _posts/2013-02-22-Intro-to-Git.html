Introduction to GIT

I've used both GIT and Subversion and personally don't have a preference. I wanted do a little write up so I decided to make this post. This tutorial goes through the git command line but you can download the tortoise GIT GUI client.

Creating a repository 


     mkdir testrepo
	 
     cd testrepo
	 
     git init

Checkout a repository

  
     git clone path or git clone [urlname]

Check the status of a branch


     git status

Check the commit history of a branch


     git log

Adding a Remote repository


     git remote add origin [urlname]

Adding a file to your local repository


     touch Readme.md
	
     git add Readme.md
	
     git commit -m "init"

Branching


     Master is the default branch
   
     git branch testbranch
	
     git checkout testbranch
  
     Switch back to master
   
     git checkout master
   
     Shorthand for the above 
   
     git checkout -b testbranch

Pushing to a remote branch 


     git push

Merging


     git merge nameofbranch
