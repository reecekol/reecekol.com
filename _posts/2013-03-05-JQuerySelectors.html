JQuery Selectors

Selectors are at the heart of JQuery and are part of what makes the library so powerful. John Resig designed JQuery so that developers could leverage their existing knowledge of CSS selectors to to manipulate the DOM.Here are a few common selectors.

JQuery Generic Syntax


  
  A typical JQuery statement follows the syntax


  $("selector").method();
 
  
  //calls the focus method on the DOM element with and id of helloworld

  
  $("#helloworld").focus();

Select an HTML Element or its Descendant

   
  
  $("element").method();

  $("p").css("color","red")

  
  // descendant selectors in the dom 

  
  $("element1 element2").method()

Select a direct child element of element1 


  
  // matches direct children of element1 

  
  $("element1 > elmenet2").method() 

  
  //selects paragraph elements that are direct children of the div element

  
  $("div > p ").method()

Universal Selector


  
  //univiersal selector selects all elements

  
  $("*").method()

Elements with specific ids and classes 


  
  //Element with a class of classname

  
  $("element.classname").method();  

  
  //Element with a id  of idname 

  
  $("element#idname").method();

Select Elements with a particluar attribute 

     
  //selects and element with and attributename of attribute

	 
    $("element[attribute]").method();
