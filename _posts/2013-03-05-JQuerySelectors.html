JQuery Selectors

Selectors are at the heart of JQuery and are part of what makes the library so powerful. John Resig designed JQuery so that developers could leverage their existing knowledge of CSS selectors to to maniupatlate the DOM.Here are a few common selectors.

JQuery Generic Syntax


  
A typical Jquery statement follows the syntax


  $("selector").method();
 
  
//calls the focus method on the DOM element with and id of helloworld

  
  $("#helloworld").focus();
